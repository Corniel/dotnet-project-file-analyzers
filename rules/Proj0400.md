# Proj0400: Define the project publishability explicitly
To prevent confusion, explicitly define the
`<IsPublishable>` node.

## Non-complaint
``` XML
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFramework>net7.0</TargetFramework>
  </PropertyGroup>

</Project>
```

## Compliant
``` XML
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFramework>net7.0</TargetFramework>
    <IsPublishable>true</IsPublishable>
  </PropertyGroup>

</Project>
```
